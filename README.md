# Website Hamzah RPL

Repositori ini berisi website portofolio sederhana siap-publish ke **GitHub Pages**.

## Struktur folder
```
website-hamzah-rpl/
├── index.html
├── style.css
├── script.js
├── img/
│   └── profil.jpg
└── README.md
```

## Fitur
- Tema gradasi biru–hijau–ungu
- Mode terang / gelap (disimpan menggunakan `localStorage`)
- Animasi masuk (fade & slide)
- Popup notifikasi saat mengirim pesan
- Responsif untuk perangkat mobile dan desktop

## Cara cepat menjalankan (offline)
1. Ekstrak `website-hamzah-rpl.zip`.
2. Buka file `index.html` di browser (klik dua kali atau buka via browser).

## Cara publish ke GitHub Pages (panduan singkat)
1. Buat repositori baru di GitHub (mis. `website-hamzah-rpl`).
2. Upload seluruh isi folder (`index.html`, `style.css`, `script.js`, `img/`, `README.md`) ke repositori.
3. Buka tab **Settings** di repositori GitHub.
4. Pilih **Pages** (atau "Pages" di menu sebelah kiri).
5. Pada bagian "Source", pilih `main` branch (atau `master`) dan folder `/ (root)`, lalu klik **Save**.
6. Tunggu beberapa menit — GitHub akan memberikan URL seperti: `https://<username>.github.io/<repo-name>/`.
7. Buka URL tersebut untuk melihat website online.

## Mengganti foto profil
Ganti file `img/profil.jpg` dengan foto milikmu (ukuran ideal 400x400px), nama file tetap `profil.jpg`.

Jika butuh saya bantu upload ke GitHub Pages secara langkah-demi-langkah, beri tahu saya username GitHub-mu dan saya bantu panduannya.
